## 内容

文字列を計算する電卓作成ハンズオン

```
1 2 + = 3 
1 2 - = -1 
6 1 2 + / = 2 
1 2 3 * + = 7 
1 2 3 * + 10 * = 70 
1 2 3 * + 10 * 7 / = 10 
1 2 3 * + 10 10 * + = 107 
1 2 3 * - 10 10 * - = -105 
```
